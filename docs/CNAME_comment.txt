Этот файл нужен для GitHub Pages.

Он должен быть в корне сайта (не репозитория, а именно по соседству
с robots.txt и index.html).

Если сайта с именем домена там не окажется, GitHub Pages забудет, что сайт
привязан к стороннему домену.

Поскольку Hugo у нас перестраивает весь веб-сайт, важно этот файл положить
прямо сюда, в статические.
